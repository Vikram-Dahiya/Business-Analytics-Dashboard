# Business-Analytics-Dashboard
Interactive dashboard built in **Microsoft Power BI** after completing the Maven Analytics Udemy course.  
It transforms raw datasets into actionable insights for sales performance, customer behavior, and KPI tracking.

## Features
- Data cleaning & transformation with **Power Query**  
- Data modeling & advanced calculations using **DAX**  
- Interactive visualizations with slicers, drill-throughs, and KPIs  
- Business insights on revenue trends, segmentation, and performance metrics
